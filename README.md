md2man - convert markdown to manpage
====================================

`md2man` is a wrapper of `pandoc` to convert markdown to manpage.

Install
-------

```sh
git clone https://github.com/weakish/md2man.git
cd md2man
make
```

Compatible with both GNU make and BSD make.

Uninstall
---------

```sh
cd md2man
make uninstall
```

Usage
-----

Command line usage:

```sh
man 1 md2man
```

Syntax:

```sh
man 5 md2man
```
